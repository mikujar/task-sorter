# 更新日志 / Changelog

## v1.0.0 (2026-03-13)

### 新功能 / Features

- ✨ **拖拽排序** - 支持卡片在收集箱和排序区之间自由拖拽，同一区域内可拖拽调整顺序
- 📥 **收集箱** - 存放未排序的任务
- 📋 **排序任务** - 已排序的任务，用箭头连接显示顺序关系
- ⏱️ **时间轴** - 显示已完成的任务及完成时间
- ➕ **添加任务** - 支持添加任务标题和描述
- ✅ **完成任务** - 点击完成按钮将任务移至时间轴
- 🗑️ **删除任务** - 支持删除任何状态的任务
- 💾 **数据持久化** - 使用 localStorage 保存数据，刷新页面不丢失
- 🌐 **中英文切换** - 支持界面语言切换，偏好会被保存
- 📱 **响应式设计** - 适配桌面端、平板和手机

### 技术栈 / Tech Stack

- React 19 + TypeScript
- Vite 8
- @dnd-kit (拖拽库)
- CSS (原生样式)
